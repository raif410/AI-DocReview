# 📁 Анализ документации из папки C:\review

## Быстрый старт

1. **Положите ваши файлы** в папку `C:\review`
   - Поддерживаются файлы: `.md`, `.txt`
   - Можно положить несколько файлов - они все будут проанализированы

2. **Запустите скрипт:**
   ```bash
   python scripts/cli/review_from_folder.py
   ```

3. **Результаты** будут сохранены в `C:\review\review_report.md`

## Как это работает

1. Скрипт читает все `.md` и `.txt` файлы из `C:\review`
2. Объединяет их в один документ
3. Отправляет на анализ через API
4. Ждет завершения анализа
5. Сохраняет отчет в `C:\review\review_report.md`

## Пример использования

```bash
# 1. Положите файлы в C:\review
#    Например: C:\review\architecture.md
#             C:\review\api.md

# 2. Запустите скрипт
python scripts/cli/review_from_folder.py

# 3. Дождитесь завершения
# 4. Откройте C:\review\review_report.md
```

## Настройка

Если хотите изменить папку, отредактируйте в `review_from_folder.py`:

```python
REVIEW_FOLDER = r"C:\review"  # ← Измените на нужную папку
```

## Требования

- Сервер должен быть запущен: `python run.py`
- Файлы должны быть в формате `.md` или `.txt`
- Файлы должны быть в кодировке UTF-8
