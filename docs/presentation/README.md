# Презентации проекта

## Презентация для HighLoad++ 2024

### Файлы

- **highload-2024.md** - Полная презентация в формате Markdown

### Конвертация в другие форматы

#### В PowerPoint (PPTX)

1. Использовать [pandoc](https://pandoc.org/):
```bash
pandoc highload-2024.md -o highload-2024.pptx
```

2. Или использовать онлайн-конвертеры:
   - [Markdown to PPTX](https://www.markdowntopresentation.com/)
   - [Dillinger](https://dillinger.io/)

#### В PDF

```bash
pandoc highload-2024.md -o highload-2024.pdf --pdf-engine=xelatex
```

#### В HTML (для веб-презентации)

Использовать [reveal.js](https://revealjs.com/):

```bash
pandoc highload-2024.md -o highload-2024.html -t revealjs -s
```

### Структура презентации

1. **Введение** (слайды 1-3)
   - О докладчике
   - Agenda
   - Проблема

2. **Решение** (слайды 4-11)
   - Архитектура системы
   - 4 компонента агентов
   - Архитектурные принципы

3. **Технические детали** (слайды 12-20)
   - Технологический стек
   - Высоконагруженные аспекты
   - Оптимизации

4. **Результаты** (слайды 21-25)
   - Метрики
   - Вызовы и решения
   - Roadmap

5. **Заключение** (слайды 26-30)
   - Выводы
   - Вопросы

### Рекомендации для доклада

- **Длительность**: 30-40 минут + 10 минут Q&A
- **Фокус**: Высоконагруженные аспекты, производительность, масштабируемость
- **Аудитория**: Инженеры, архитекторы, DevOps специалисты

### Ключевые моменты для выделения

1. **10,000 транзакций/сек** - реальная производительность
2. **<100ms latency (p95)** - достигнутые показатели
3. **>99.9% доступность** - надежность системы
4. **Автоматическое разрешение >70% инцидентов** - эффективность AI
