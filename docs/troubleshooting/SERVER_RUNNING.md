## ✅ Как проверить, что сервер работает

### Способ 1: Откройте другой терминал

1. **Откройте новый терминал** (PowerShell или CMD)
2. **Выполните:**
   ```bash
   curl http://localhost:8000/health
   ```
   
   Должен вернуться: `{"status":"healthy"}`

### Способ 2: Откройте в браузере

Откройте: **http://localhost:8000/docs**

Вы увидите Swagger UI - это значит сервер работает!

### Способ 3: Запустите тест

В новом терминале:
```bash
python scripts/tools/test_api_quick.py
```

Если запрос проходит - сервер работает!

## 🎯 Что делать дальше

**Оставьте этот терминал открытым** - сервер должен работать в фоне.

**В новом терминале** или через Swagger UI отправьте запрос на анализ:

**Swagger UI:**
1. Откройте http://localhost:8000/docs
2. Найдите `POST /api/v1/review/start`
3. Заполните поля и отправьте запрос

**Python:**
```python
import requests

response = requests.post(
    "http://localhost:8000/api/v1/review/start",
    json={
        "document": "# Ваша документация...",
        "document_type": "markdown",
        "context": {}
    }
)
```
