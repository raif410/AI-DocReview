# ✅ Проверка работоспособности проекта

## Самый простой способ (рекомендуется)

Запустите один скрипт, который проверит всё автоматически:

```bash
python check_project.py
```

Этот скрипт проверит:
- ✅ Все модули проекта
- ✅ Конфигурацию
- ✅ Работу сервера
- ✅ API endpoints
- ✅ Создание задач
- ✅ Структуру проекта

**Время проверки:** ~5-10 секунд

## Быстрая проверка API

Если сервер уже запущен и нужно только проверить API:

```bash
python test_api_quick.py
```

**Время проверки:** ~2-3 секунды

## Что делать дальше?

### Если все проверки пройдены ✅

Проект работает! Можете использовать API:
- Документация API: http://localhost:8000/docs
- Пример использования: `python test_example_simple.py`

### Если есть ошибки ❌

1. **Сервер не запущен?**
   ```bash
   python run.py
   ```

2. **Нет API ключа?**
   - См. `SETUP_API_KEY.md`

3. **Ошибки импорта?**
   ```bash
   pip install -r requirements.txt
   ```

4. **Другие проблемы?**
   - См. `HOW_TO_CHECK.md` для подробной диагностики

## Альтернативные способы проверки

### Ручная проверка через curl

```bash
# Проверка health
curl http://localhost:8000/health

# Должен вернуть: {"status":"healthy"}
```

### Проверка через Python

```python
import requests
response = requests.get("http://localhost:8000/health")
print(response.json())  # {"status": "healthy"}
```
